# File Integrity Checker

## Overview
A command-line tool for computing and verifying cryptographic hashes of files, implementing SHA-256 and SHA-512 algorithms using OpenSSL. This tool provides robust file integrity verification through cryptographic hashing, suitable for security-sensitive applications.

## Features
- ✅ **Dual Algorithm Support**: SHA-256 (default) and SHA-512
- ✅ **Streaming Processing**: 4KB chunks for constant memory usage with large files
- ✅ **Integrity Verification**: Compare computed hashes against expected values
- ✅ **Comprehensive Error Handling**: Graceful handling of file I/O and cryptographic errors
- **User-Friendly Interface**: Color-coded output and clear usage instructions
- **Input Validation**: Robust validation of algorithms and hash formats
